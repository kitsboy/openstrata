# OpenStrata — Investor / Partner One-Pager

            **Live:** [https://openstrata.giveabit.io](https://openstrata.giveabit.io) · **GitHub:** [https://github.com/kitsboy/openstrata](https://github.com/kitsboy/openstrata) · **Version:** `0.1.0` · **Status:** LIVE

            ## One sentence
            Open standard for sovereign data portability across Bitcoin, Lightning, and Nostr.

            ## Problem
            Identity/data are siloed across Bitcoin ecosystem tools with no portability standard.

            ## Solution
            Open standard + reference implementation for carrying identity/reputation/data across layers.

            ## Who it's for
            Protocol developers, wallet/app builders, sovereign users.

            ## Stage
            Live landing + standard scaffolding

            ## Traction (honest)
            Live openstrata.giveabit.io; template docs complete; workplan active.

            ## Model / value flow
            Open standard (public good); optional commercial reference services later.

            ## Why Give A Bit
            Part of an interlocking Bitcoin-sovereignty suite. Shared brand, Safe Harbour, open-source default, and cross-product rails (OTS, Lightning, Nostr) compound each product.

            ## 90-day north star (default)
